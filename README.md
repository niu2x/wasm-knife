# WASM Knife

## wasm-trim-func

	Usage: wasm-trim-func [OPTION]... [FILE]
	trim function from wasm
	  -c,                        config file, one function name per line
	  -f,                        function name[s], separated by ','
	  -h,                        show help
	  -n,                        dry run
	  -o,                        output file
	  -t,                        output WebAssembly Text
